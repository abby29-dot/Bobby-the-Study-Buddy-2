# 📚 Bobby the Study Buddy

**Live Demo:** https://projectexportzip--adutra29.replit.app/

**Bobby the Study Buddy** is an AI-powered learning assistant that helps students better understand school material through interactive support and intelligent responses. Whether you're reviewing concepts, asking questions, or reinforcing what you've learned, Bobby is designed to make studying easier and more engaging.

## 🚀 What It Does

* Answers questions about school topics
* Helps explain concepts in a clear, simple way
* Provides AI-powered learning assistance
* Supports students with studying and review

## 🛠️ Tech Stack

* Python
* TensorFlow
* Replit
